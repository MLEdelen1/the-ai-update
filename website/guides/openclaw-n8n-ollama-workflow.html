COMBINE GUIDE

Build a daily capture-to-summary workflow

Stack: OpenClaw handles commands, n8n handles flow logic, Ollama handles local text generation.

PREREQUISITES

Before you combine tools

Complete install and use guides for OpenClaw, n8n, and Ollama first.

STEP 1

Start all three services

openclaw gateway start
ollama serve
n8n start

Expected result: All services are running locally.

STEP 2

Create n8n workflow shell

In n8n, create New Workflow, add Manual Trigger, HTTP Request node, and Set node.

Expected result: You have a 3-node pipeline.

STEP 3

Call Ollama from n8n

Set HTTP Request to POST http://localhost:11434/api/generate with JSON body containing model, prompt, and stream:false.

Expected result: n8n receives local AI response JSON.

STEP 4

Format output in Set node

Map the generated text into clean fields like summary, next_action, and priority.

Expected result: Structured output you can pass to other tools.

STEP 5

Run workflow from OpenClaw session

Use OpenClaw while workflow runs to capture notes, then paste notes into n8n manual trigger input for summarization.

Expected result: One repeatable loop from raw notes to ready actions.
