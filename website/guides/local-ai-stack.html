GUIDE • FULL STACK

Local AI stack build order that avoids rework

Combine Ollama, OpenClaw, n8n, and Agent Zero in sequence with validation checkpoints after each phase.

PHASE 1

Install Ollama first

Get local inference stable before adding automation layers.

Run: ollama --version, ollama pull llama3.2:3b, ollama run llama3.2:3b "health check"
PHASE 2

Stand up OpenClaw control layer

Install OpenClaw, verify gateway lifecycle, and test one real task.

Run: openclaw help, openclaw gateway start, openclaw gateway status
PHASE 3

Add n8n automation

Trigger predictable flows, then route content through Ollama and OpenClaw.

Run: n8n --version, n8n start, then test at http://localhost:5678
PHASE 4

Enable Agent Zero for bounded autonomy

Keep strict step caps and explicit tool permissions while piloting.

Run: agent-zero config show, agent-zero run "read-only task"
STACK VERIFICATION

Exact final validation sequence

ollama list
curl http://localhost:11434/api/tags
openclaw gateway status
n8n --version
agent-zero --help
STACK TROUBLESHOOT

Known-good reset sequence

openclaw gateway stop
Stop-Process -Name ollama -Force -ErrorAction SilentlyContinue
Stop-Process -Name n8n -Force -ErrorAction SilentlyContinue
openclaw gateway start
Start-Process ollama -ArgumentList "serve"
n8n start
