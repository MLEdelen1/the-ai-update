Use Agent Zero after install

This page focuses on first practical use, not installation.

STEP 0

Confirm container is running

docker ps

Expected result: You see agent0ai/agent-zero in running containers.

STEP 1

Open Agent Zero UI

http://localhost:50001

Expected result: Interface opens in browser.

STEP 2

Run your first structured task

Paste a clear task with goal, context, and required output format.

Goal: summarize these support tickets.
Context: UHealth queue from today.
Output: top 5 issues and 1 action for each.

Expected result: Response is easier to use than a vague prompt.

STEP 3

Stop when done

docker stop $(docker ps -q --filter ancestor=agent0ai/agent-zero)

Expected result: Container stops and local resources free up.
