OpenClaw setup from install to first useful task

Who this serves: people who want one control layer for AI workflows. Next action: finish all 5 steps and run one real task.

STEP 1

Confirm prerequisites

Install Node.js LTS and verify your shell can run OpenClaw commands.

Checkpoint: openclaw help runs successfully.
STEP 2

Install and start gateway

Install OpenClaw and start gateway service with default settings first.

Checkpoint: openclaw gateway status returns running.
STEP 3

Connect one model lane

Use either one cloud model key or one Ollama local endpoint, not both at first.

Checkpoint: test prompt returns expected output.
STEP 4

Run first practical workflow

Use a real task like rewriting an email or summarizing notes.

Checkpoint: same workflow works twice with consistent quality.
STEP 5

Add safety and scale

Document prompts, set boundaries, and prepare for n8n or Agent Zero integration.

Checkpoint: one baseline workflow is documented and repeatable.
