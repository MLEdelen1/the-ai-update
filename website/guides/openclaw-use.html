Use OpenClaw after install

This page assumes OpenClaw is already installed. If not, run the install guide first.

STEP 0

Confirm install is done

openclaw help
openclaw gateway status

Expected result: Help output appears and gateway reports running.

STEP 1

Restart gateway safely

openclaw gateway restart
openclaw gateway status

Expected result: Service restarts and status returns healthy.

STEP 2

Run a simple workspace command

cd $env:USERPROFILE
mkdir openclaw-use-demo -Force
Set-Location openclaw-use-demo
openclaw help

Expected result: OpenClaw works from your project folder too.

STEP 3

Daily usage pattern

Start your day with gateway status, use OpenClaw for tasks, then stop only if you need to free resources.

openclaw gateway status
# do your work
openclaw gateway stop

Expected result: You can control exactly when OpenClaw is active.
