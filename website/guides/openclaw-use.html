USE GUIDE ? OPENCLAW

Main takeaway: use status, help, and restart commands every day

CLI means command-line interface, which is a text tool for running app commands.

WINDOWS

PowerShell quick check

openclaw gateway status
openclaw help
openclaw gateway restart

Expected result: status prints, help opens, and restart succeeds.

macOS

Terminal quick check

openclaw gateway status
openclaw help
openclaw gateway restart

Expected result: same output pattern as Windows.

Linux

Terminal quick check

openclaw gateway status
openclaw help
openclaw gateway restart

Expected result: gateway is healthy after restart.

WORKSPACE

Run from any folder

mkdir openclaw-demo
cd openclaw-demo
openclaw help

Expected result: OpenClaw works in your project folder.

TROUBLESHOOTING

Common fixes

If command is not found, open a new terminal window and try again.

If gateway hangs, run openclaw gateway stop then openclaw gateway start.

If errors continue, revisit OpenClaw setup.
