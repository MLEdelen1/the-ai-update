Use n8n after install

Build one simple workflow so you understand how triggers and nodes work.

STEP 0

Confirm n8n is running

n8n --version
http://localhost:5678

Expected result: Version appears and editor opens.

STEP 1

Create first workflow

In n8n, click New Workflow, add Manual Trigger, then add Set node.

Expected result: You see two connected nodes.

STEP 2

Add sample output

In Set node, create a field named message with value: Hello from n8n.

Expected result: Output data appears when you run manually.

STEP 3

Run and save

Click Execute Workflow, confirm output, then Save and name it First Workflow.

Expected result: You now have a reusable automation base.
