INSTALL GUIDE • AGENT ZERO

Install Agent Zero with Docker prerequisite checks on every OS

Official quick-start commands plus beginner troubleshooting.

BEGINNER GLOSSARY

Quick words

Model: the AI brain version. Local: runs on your own computer. API: a way apps send messages to each other. Workflow: repeatable steps that do one job. Token: small pieces of text a model reads and writes.

OFFICIAL SOURCE

Where commands come from

Commands below come from the Agent Zero official README quick start: https://github.com/frdel/agent-zero.

PREREQUISITE

Install Docker first

Windows/macOS: open Docker Desktop download, install, then open Docker Desktop and wait until it says running. Linux: install Docker Engine from your distro docs.

docker --version

Expected result: Docker version prints.

If this fails, do this: Restart machine after Docker install and run version command again.
WINDOWS + MACOS + LINUX

Run Agent Zero container

docker pull agent0ai/agent-zero
docker run -p 50001:80 agent0ai/agent-zero

Open browser to http://localhost:50001.

Expected result: Agent Zero web UI opens.

If this fails, do this: Run docker ps. If no container is running, run docker run again. If port is busy, use docker run -p 50002:80 agent0ai/agent-zero and open port 50002.
POST-INSTALL CHECK

Verification

docker ps
docker images | findstr agent-zero

Expected result: You see running container and downloaded image.
