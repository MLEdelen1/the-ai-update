FREE STARTER KIT

The 10-Minute AI Workflow Starter Kit

Use these copy-ready prompts with real examples so you can get value in minutes, not after a weekend of setup. Start with one workflow that matches your current bottleneck, run it daily for a week, then keep the one that saves you the most time.

WORKFLOW 1

Clear Message Rewriter

Prompt: Rewrite this message to be clear, polite, and concise. Keep meaning and add one specific next step with a deadline.

Example input: “Hey, just checking in again because this has been sitting and we really need it done soon.”
Example output: “Hi team, following up on the access request. Could you complete this by Thursday at 3 PM so we can stay on schedule?”
WORKFLOW 2

Notes to Action Plan

Prompt: Turn these meeting notes into decisions, owners, due dates, and blockers. Keep it to one short table and one risk summary.

Example input: onboarding docs outdated, support queue high, waiting on API key.
Example output: Update onboarding docs (Avery, Fri), reduce queue by canned replies (Sam, Wed), request API key from IT (Jordan, today), risk: launch slips if key not approved.
WORKFLOW 3

Daily Focus Planner

Prompt: Pick top 3 priorities by urgency and impact, then produce a realistic 4-hour execution block with break timing.

Example tasks: reply to 18 tickets, finalize invoice, prep Monday report, clean notes.
Example result: Priority 1 tickets (90 min), Priority 2 invoice (45 min), Priority 3 report outline (60 min), admin cleanup (30 min), plus 15-minute buffer.
WORKFLOW 4

Follow-Up Builder

Prompt: Write three follow-ups for this context: 24 hours, 72 hours, and 7 days. Keep tone respectful and direct, and include one CTA in each.

Use case: client has not approved proposal.
Result: Day 1 reminder, Day 3 value-focused nudge, Day 7 close-the-loop message with clear yes/no ask.
WORKFLOW 5

Weekly Review

Prompt: Analyze this week’s log. Return wins, misses, root causes, and the top 3 changes for next week. Keep it practical and measurable.

Example outcome: Win: resolved 42 tickets, Miss: late escalations, Root cause: no triage window, Change: add 2 daily 20-minute triage blocks and enforce escalation tags.
