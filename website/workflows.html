WORKFLOW LIBRARY

Real workflows you can run in 10–20 minutes

This page is for practical repeatable tasks. Each workflow has a trigger, a clear output, and a decision you can act on right away.

⚡

Clear Message Rewriter

Use when: an email, ticket, or DM is messy or emotional.

Output: one concise version, one friendly version, one firm version.

📋

Notes to Actions

Use when: you have brain-dump notes and no clear plan.

Output: action list with owner, due date, and priority.

🎯

Priority Planner

Use when: you have too many tasks and limited time.

Output: top 3 priorities, what to defer, and a realistic schedule.

🔄

Follow-Up Builder

Use when: conversations stall and people go quiet.

Output: short follow-up sequences for 24h, 72h, and 7-day check-ins.

📊

Weekly Review

Use when: you need to improve execution week over week.

Output: wins, misses, root causes, and next-week corrections.

🧠

Decision Brief

Use when: you are stuck between 2–3 options.

Output: tradeoff matrix, risk notes, and a recommended next move.
