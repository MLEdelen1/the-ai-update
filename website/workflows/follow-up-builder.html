WORKFLOW

Follow-Up Builder

Create polite, persuasive follow-up sequences that keep momentum without sounding pushy.

What this solves

This workflow gives you structured follow-up messages with timing and tone changes so important asks do not die in inbox silence.

When to use / when not to use

  • Use after proposals, sales calls, interview loops, or unresolved requests.
  • Do not use when the recipient has already declined clearly.

Inputs needed

  • Original message or proposal summary
  • Recipient role and relationship
  • Desired outcome and deadline

Copy-ready prompt block

Build a follow-up sequence for a stalled conversation.

Need three messages:
- 24-hour nudge
- 72-hour value add
- 7-day close-the-loop

Rules:
- Keep each under 90 words
- Include one clear CTA each time
- Increase directness slightly at each step
- Keep tone respectful and calm

Context:
[paste original context]

Execution steps

  1. Feed the AI the original context and recipient type.
  2. Review each step for tone and clarity.
  3. Schedule all three messages in your CRM or reminders app.
  4. Stop sequence immediately once a reply is received.

Expected output format

Day 1 (24h):
Subject:
Message:
CTA:

Day 3 (72h):
Subject:
Message:
CTA:

Day 7:
Subject:
Message:
CTA:

Concrete example

For an unanswered proposal, day-1 asks if there are questions, day-3 shares a short case study, and day-7 asks for a yes, no, or later date to close the loop cleanly.

Common mistakes

  • Sending the same wording three times
  • Writing long follow-ups without a CTA
  • Not changing cadence based on urgency

Quick quality check

  • Each message has one ask
  • Tone is respectful, not needy
  • Value is added by message two
  • Final message gives an easy out
